1.6、Component.event（自定义事件）

Component.event：Regular中，事件分为两类：1、DOM事件 2、组件事件

自定义DOM事件:可以注册一些dom原生并不支持的事件，比如on-tap, on-hold.

注意regularjs遇到on-xx属性时，会首先去检查是否注册了xx这个自定义事件，如果没有，无论这个事件浏览器是否支持，都会走简单DOM事件的流/程（即都会通过addEventListener来绑定事件）

Arguments
event: 自定义事件名 (没有on-前缀)
fn(elem, fire)
elem: 绑定节点
fire: 触发器
注意如果需要做 销毁工作 ，与指令一样，你需要返回一个销毁函数

Example :

                                    
				//自定义DOM事件

				var Compon = Regular.extend({
					template:'',
				});

				
				//自定义一个  回车键  事件，这个达到的效果是：当按下 回车键 时，执行submit函数！
				var dom=Regular.dom;

				//自定义 回车键 事件
				Compon.event('enter',function(ele,fire){

					function update(ev){

						if(ev.which == 13){

							ev.preventDefault();
							fire(ev);
							alert(121);
						}
					}
					
				//让自定义的 回车键 事件，按下键盘的时候触发！

				dom.on(ele,"keypress",update);

					return function destroy(){ 

					//这里返回一个用于销毁工作的销毁函数，不需要销毁函数时可以不用返回

						dom.off(ele,"keypress",update);
					}
				});

		//注意：compon要在 自定义事件 后面（定义）执行，看好执行顺序！

		//这里的执行顺序是先执行submit函数（弹出 88888），在执行update函数中的内容（这里弹出121）.
				var compon = new Compon({

					submit:function($event){

						alert(88888);
					}

				});

				//compon.$inject('#app');
实例演示： 基本dom事件及自定义事件的理解

2、实例接口

component即代表组件实例, 注意这些公有都有$前缀 意味不建议进行重写。

2.1、component.$inject

component.$inject(element[, direction])：插入组件到指定位置

Example :

                                    
						//$inject的基础用法

						var Component = Regular.extend();

						//这里是模板加载的另一种方法（template）

						var component1 = new Component({

							template:'
{title}
',

							data:{title:'Example1'}

						});
						component1.$inject('#aapp');//默认添加到子元素末尾位置

						/////////////////////////////////////////

						var component2 = new Component({

							template:'
{title}
',

							data:{title:'Example2'}

						});
						component2.$inject('#aapp','top');//添加到子元素的首位

						/////////////////////////////////////////

						var component3 = new Component({

							template:'
{title}
',

							data:{title:'Example3'}
						});
						component3.$inject('#aapp','after');//添加到#app元素下边兄弟节点的位置

						/////////////////////////////////////////
						var component4 = new Component({

							template:'
{title}
',

							data:{title:'Example4'}
						});

						//component4.$inject('#aapp','before');//添加到#app元素上边兄弟节点的位置
实例演示： $inject的基础用法

component.$refs：在模板中，你可以使用ref属性来标记一个节点或组件.　在实例化后，你可以通过component.$refs 来获取你标记的节点

2.6、component.$on\$off\$emit

component.$on(event, fn]):事件绑定；如果event参数传入一个Object, 会成为一个多重事件绑定。
component.$off([event] [,fn])：1.如果同时传入 event和fn,　则移除指定event类型下的fn函数。
2.只传入event, 移除所有event对应的监听器。
3.什么都不传，移除所有。
component.$emit(eventName [, args...])：触发指定事件。
